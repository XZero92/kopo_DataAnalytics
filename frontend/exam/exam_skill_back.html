백엔드 개발에 특히 관심이 많아, 이 분야에 필요한 여러 기술을 깊이 있게 학습하고 있습니다. 제가 주로 사용하는 백엔드 기술은 Python과 Java입니다. 이 언어들을 사용하여 서버 사이드 로직과 데이터베이스 관리를 합니다. Node.js와 Express 프레임워크를 통해 비동기 처리와 미들웨어의 활용 방법을 배우며, RESTful API 설계 방법을 실습했습니다. 또한, MongoDB와 같은 NoSQL 데이터베이스를 사용하여 대용량 데이터 처리와 비구조적 데이터의 효율적 관리 방법을 경험했습니다. 데이터베이스 관리 시스템으로는 MySQL과 PostgreSQL을 사용해보았고, 이를 통해 SQL 쿼리 최적화와 트랜잭션 관리 기술을 개발했습니다. Docker와 같은 컨테이너 기술을 사용하여 어플리케이션의 배포 및 관리를 더욱 효율적으로 할 수 있게 되었습니다. 이러한 기술들을 통해 서버의 성능을 최적화하고 보안을 강화하는 방법을 배웠으며, 클라우드 서비스(AWS, Azure)를 이용한 환경 구축에도 참여했습니다. 앞으로도 체계적인 백엔드 시스템 구축과 성능 최적화를 위해 지속적으로 학습하고, 전문 백엔드 개발자로 성장하겠습니다.
